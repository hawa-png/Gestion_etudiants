# Application de bonjour avec nodejs et express 

##  Application de bonjour developpée avec nodejs et express. 

## Technologies

- nodejs
- express

## Structure du projet

Gestion_etudiants/
  server.js
  package.json
  package-lock.json


## Prerequis

- nodejs version 18+


## Vérification  

- node -v
- npm -v 

## Installation 

### 1. Cloner le depot 

git clone https://github.com/hawa-png/Gestion_etudiants.git

### 2. Se deplaçer dans le dossier 

cd Gestion_etudiants.git

### 3. Installations des dependances 

npm install 

### 4. Lançer le projet 

node server.js
